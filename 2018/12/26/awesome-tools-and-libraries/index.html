Awesome

发表于 2018-12-26 | 更新于 2019-11-10 | 分类于 工具 |

近期在github上看到了Awesome Go/Python/PHP/Java系列，对一些常用的框架和库做出整理，鉴于自己也是一个深度工具依赖着，也想对自己收集到的一些开发者常用工具进行整理。

Awesome汇总

整理了github上所有的awesome系列

API文档管理

  • Slate ：开源，单页面的静态API展示工具

  • docsify : 将md格式文件以Web形式渲染，简单易用，扩展性强，通常用于文档展示，类似的开源产品有Vuepress和Gitbook

  • Sphinx ：老牌开源，基于Python，生成静态HTML页面

  • Apizza ：国产，收费，类似Postman，便于团队协作

    绘图

    • ProcessOn：在线绘制流程图，时序图等
    • StarUML：Mac上画图工具，支持UML等多种类型

Java

  • 故障排查
    • gceasy：在线GC日志分析工具，点击直达
    • fastthread：在线的Thread Dump文件分析，点击直达
    • Arthas：阿里开源的诊断工具，支持通过在线修改Class来定位问题，简单易用，文档入口

Rust

  • Distributed Solution

    • 分布式事务
      • Seeta：一站式分布式事务解决方案，提供多种易用的事务模型：AT 模式、TCC 模式、Saga 模式和 XA 模式
    • Database
      • Shardingsphere：数据库代理中间件，提供了数据分片，读写分离，柔性事务（Saga）等特性，Github地址
